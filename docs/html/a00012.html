Detailed Description

Implements a very basic authentication mechanism based on the setting.ini file. A simple mapping of users (and their password hashs) to groups is supported. Also mapping multiple groups into a greater one is possible.

Author:
Knut Ahlers <knut@ahlers.me>

Returns the current logged in user or null when no user is discovered.
