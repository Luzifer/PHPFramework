Welcome to this tiny PHP-framework

I'm glad you made it here. This means you already have a running instance of this framework. Now, this is just a default site to show you something other than an error message on your first visit.

Things you should try now:

  1. Look at the configuration at /config/settings.ini
  2. Define your own URL handler at /config/urls.php
  3. Write your own handler class and place them to /handler/
  4. Modify the existing or write your own templates in /templates/
  5. Place your static resources somewhere in /public/

If you have done all of that points I'm pretty sure you got how this framework is supposed to work. Now you can build nearly everything with this. You are missing something? Just implement it!
